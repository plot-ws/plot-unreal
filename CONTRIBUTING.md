# Contributing to plot-unreal

This repository is a **read-only mirror** of the `sdks/unreal` directory in
the Plot monorepo. The mirror is generated automatically — **code changes pushed
here are overwritten.**

## How to contribute

- **Found a bug or have a feature request?** Please [open an issue](https://github.com/plot-ws/plot-unreal/issues)
  here — issues are watched and triaged.
- **Want to change the code?** The source lives upstream in the Plot monorepo.
  Open an issue describing the change and we'll coordinate the upstream PR. The
  generated protocol files in this repo are produced from a shared schema and
  should not be hand-edited.

## Help & docs

- Docs: https://docs.plot.ws/sdks/unreal
- Community & questions: https://discord.gg/plot
